关于

AI Video Models Directory 是一个专注于收录和评估 AI 视频生成模型的目录网站。我们致力于为创作者、开发者和企业提供最全面、最客观的 AI 视频工具信息。

收录标准

  • 视频生成质量
  • 模型性能与速度
  • 易用性与用户体验
  • 定价与性价比
  • API 支持情况

数据来源

我们的数据来源于公开资料、官方文档、用户反馈以及实际测试。模型排名和评估基于综合性能表现，不代表任何商业合作关系。

更新频率

我们每月更新一次数据，确保信息的时效性。如有重大版本更新或新模型发布，将视情况及时调整。

联系我们

如有建议或纠错，欢迎通过 GitHub 提交 Issue 或 Pull Request。
